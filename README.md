## Overview

This project is a full-stack portfolio for TMB for managing bookings, reviews, newsletter subscriptions, and contact submissions. It includes:

- Admin Dashboard for managing bookings, reviews, contacts, and newsletter subscribers.
- Reviews System with pending approval before publishing.
- Bookings System with email confirmation for both admin and user.
- Newsletter Subscription with duplicate prevention.
- Contact Form submissions with email notifications.
- Project reviews

This project is private and proprietary. Unauthorized use, copying, or distribution is prohibited.

---

## Features

- Admin login and dashboard
- CRUD operations for reviews, bookings, and contacts
- Email notifications for bookings, reviews, and contact submissions
- Newsletter subscription with validation
- Pending approval system for reviews
- Data visualizations for counts (bookings, reviews, contacts, subscribers)
- Project reviews and Tech Stack

---

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS, Vite 
- **Backend:** Node.js, Express  
- **Database:** MySQL  
- **Email:** Nodemailer (SMTP)  
- **Environment Variables:** dotenv
